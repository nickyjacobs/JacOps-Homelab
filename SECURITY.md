# Security Policy

This repository documents a personal homelab. It does not ship production code. Still, if you find something in the documentation that could be abused (leaked credentials, exposed endpoints, private keys), please report it privately.

## Reporting

Open a GitHub security advisory or reach out via [LinkedIn](https://www.linkedin.com/in/N-O-Jacobs). Do not open a public issue for sensitive findings.

## Scope

- Leaked secrets in the repository history
- Accidentally published private IPs, domains or credentials
- Outdated advice that could harm readers who follow it

## Not in scope

- Vulnerabilities in third party tools mentioned in the docs (report those upstream)
- Design preferences or opinions
